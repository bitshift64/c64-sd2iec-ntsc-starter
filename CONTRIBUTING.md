## Submitting new sources

New file sources are welcome, but they must be legal to download and appropriate for this project.

Good candidates include:

- Public domain software
- Freeware or open-source releases
- Author-hosted downloads
- Official project download pages
- NTSC or PAL/NTSC confirmed C64 files
- Files known to work on SD2IEC, or clearly marked as untested

Please do not submit:

- Commercial game dump collections
- Cracked commercial game packs
- Random archive mirrors with unclear permissions
- PAL-only files unless they are clearly useful and marked as such
- Files requiring true 1541 emulation unless clearly documented

Preferred process:

1. Open a GitHub Issue using the “New source request” template.
2. Include the source URL, compatibility notes, and licensing/distribution details.
3. If possible, test the file on real NTSC C64 hardware with SD2IEC.
4. Maintainers will review the source before adding it to the downloader.

Sources may be rejected if the licensing is unclear, if the file appears to be commercial software, or if it does not fit the NTSC SD2IEC focus of the project.
