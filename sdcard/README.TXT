C64 SD2IEC NTSC CLEAN CARD V13

This version is strict about C64 friendly filenames.
Visible C64 folders and downloaded content use short uppercase 8.3 style names where possible.
That avoids the odd PETSCII/long filename characters shown by many SD2IEC file browsers.

V13 also tries to place FB first in the root directory order after downloading.
That is for users who want to start FileBrowser with:

LOAD"*",8
RUN

IMPORTANT FRESH INSTALL NOTE:
For best results, do not simply overlay this on an old card.
Format the SD card or delete the old generated folders first, then extract this zip to the card root.
Old long filenames will remain if you only copy this over the previous card.

After extracting, run this from Windows:

PCFILES\RUN.BAT

On the C64, preferred quick start:
LOAD"*",8
RUN

Direct FileBrowser start also works:
LOAD"FB",8
RUN
