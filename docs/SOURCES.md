# Sources

This project is a downloader and folder-layout helper. It intentionally avoids bundling commercial game collections.

The downloader currently references:

- CBM FileBrowser from TheFutureWas8Bit.
- DirPlus from Bo Zimmerman / Boray.
- TheFutureWas8Bit SD2IEC software pack.
- Scene World diskmag issue downloads.
- Zimmers NTSC-fixed C64 demos.
- Zimmers freely distributable C64 games.
- Optional CSDb music/demo sample, filed under `MUSIC\ADBDANCE`.

Downloaded third-party files remain under their original authors' and distributors' terms.
