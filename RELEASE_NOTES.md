# Release Notes

## v0.1.2-alpha / card package v13

- Tones down the README so the project reads as a general NTSC SD2IEC starter card, not as a spotlight on any one included sample.
- Keeps the optional music/demo sample under `MUSIC\ADBDANCE` for users who want it.
- Keeps `FB` boot-first behavior for `LOAD"*",8`.
- Keeps short uppercase C64-visible filenames and 8.3-friendly downloaded content.
- Keeps 7-Zip support for ZIP, GZ, and 7Z extraction.

## v0.1.1-alpha / card package v12

- Moves the optional music/demo sample from a dedicated root folder to `MUSIC\ADBDANCE`.
- Removes the dedicated root folder so the public package feels more general-purpose.
- Keeps `FB` boot-first behavior for `LOAD"*",8`.
- Keeps short uppercase C64-visible filenames and 8.3-friendly downloaded content.
- Keeps 7-Zip support for ZIP, GZ, and 7Z extraction.

## v0.1.0-alpha / card package v11

- Initial public packaging candidate.
- Clean short-name root layout.
- CBM FileBrowser `FB*` variants copied to root.
- Boot-first finalizer for `LOAD"*",8`.
- Scene World, Zimmers NTSC-fixed demos, selected public-domain games, DirPlus, and FileBrowser download support.
